Operating System Questions and Answers

What are the characteristics of a file?

Files may be characterized by (1) Volatility (2) Activity and (3) Size.

What is a thread?

A thread, sometimes called a lightweight process (LWP), is a basic unit of CPU utilization; it comprises of a thread ID, a program counter, a register set and a stack.

What is multithreading?

A multithreaded process contains several different flows of control within the same address space. The benefits of multithreading include increased responsiveness to the user, resource sharing within the process, economy and the ability to take advantage of multiprocessor architectures.

What are the basic services provided by an operating system?

(a) Program execution
(b) I/O operations
(c) File system manipulation
(d) Communications
(e) Error detection

What are system calls?

It provides the interface between a process and the operating system. These calls are generally available as assembly-language instructions, and they are usually listed in the various manuals used by the assembly-language programmers.

List the types of system calls.

(a) Process control
(b) File management
(c) Device management
(d) Information maintenance
(e) Communications.

What are the features of the UNIX operating system?

Portability, Machine-independence, Multi-user operations, Hierarchical file system, UNIX shell, Pipes and Filters, Utilities and Background processing.

What type of an operating system is the UNIX?

The UNIX operating system is a multi-programming, time-sharing, and multi-tasking system.

What are the components of UNIX?

Kernel, UNIX utilities and application software are the components of the UNIX.

List the types of files in UNIX.

Ordinary files, Directory files and Special files.

List the various types of users in UNIX

System administrator, File owner, Group owner and other users.

What is a Shell?

A Shell is an intermediary program which interprets the commands that are typed at the terminal, and translates them into commands that the kernel understands.

List the various types of shells.

The shell runs like any other program under the UNIX system. Some of the popular shells are: Bourne shell, C shell, Korn shell and restricted shell.

What is a Pipe?

Pipe is a feature by which the standard output of a command or user program can be sent as the standard input to another command or user program.

What is the use of the echo command?

The echo command is used to display messages on the screen.
