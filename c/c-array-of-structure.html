Array of Structures

Array of structures

  • To display any kind of bills for several customers, then we need several structures. In such cases, array of structures will be useful to us.We can declare array of structure like this:

    struct store st[50];

    Here, we have declared 50 structure variables starting from st[0] to st[49]. In this, st[0] represents the first customer's electricity bill, st[1] represents the second customers electricity bill etc. In general we can write st[i] to access the elements, we can write. 

    st[i].mno;
st[i].name;
st[i].previous;
st[i].present;
st[i].units;
st[i].charge;
