strcpy() and strncpy()

  • strcpy() and strncpy() functions are useful to copy a group of characters from one string into another string.

  • Let us take s1 as an empty string and s2 as a string with a group of characters.

    strcpy(s1, s2);

    This will copy all the characters from s2 into s1. Suppose, we do not want all the characters from s2 to be copied, then we can use strncpy(). This function is useful to copy only a specified number of characters from s2 into s1.

    strncpy(s1, s2, n);

    Here, the first n characters from s2 will be copied into s1.

    • Program to copy a few characters from s2 into s1

    #include<stdio.h>
#include<string.h>
void main()
{
   char s1[10], s2[50];
   int n;

   printf("\n Enter string: ");
   gets(s2);

   printf("How many characters to copy??");
  scanf("%i",&n);

  strncpy(s1,s2,n);
  printf("\n The first %d chars are: %s", n,s1);
}

    Output:

    Enter a string: Navtutorials
How many chars to copy??
3
The frist 3 characters are: Nav
